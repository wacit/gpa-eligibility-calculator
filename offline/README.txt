GPA & Athletic Eligibility Calculator — Portable (Offline) Version
==================================================================

QUICK START
-----------

Windows:
  Double-click  Launch.cmd
  (Or right-click Launch.ps1 -> Run with PowerShell)

macOS:
  Double-click  Launch.command
  (First time: right-click -> Open, to bypass Gatekeeper warning.)

Linux / anywhere:
  Just open  app/index.html  in any modern browser
  (Chrome, Edge, Firefox, Safari — anything from 2022 onward).


WHAT WORKS OFFLINE
------------------
  - GPA calculation on 7-point and 10-point scales (weighted + unweighted)
  - NCAA Division I & II core-course GPA check
  - NAIA "2 of 3" eligibility (GPA + ACT/SAT + class rank)
  - Senior-year projection (planned courses)
  - Save / load profile to a file
  - Paste / AI import (the parser; the AI prompt copy still works)
  - Light / dark mode

WHAT NEEDS INTERNET
-------------------
  - College search & match (US College Scorecard live API)
  - Buy Me a Coffee widget
  - Sending the AI prompt to ChatGPT/Claude/Gemini (those run on their servers)


PRIVACY
-------
All grade/score entry stays in your browser. The online live URL at
https://gpatelca.com is the same code as in this folder. There is no
account, sign-up, or analytics in this app.


UPDATING
--------
This portable version is a snapshot. To get the latest version, redownload
the zip from https://gpatelca.com/download/ (or replace the "app" folder
with a fresh copy from the repo).
